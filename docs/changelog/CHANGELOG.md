# Changelog

All notable changes to **Aether Architect** will be documented in this file.

## [3.0.0] - 2024-05-20
### Added
- **Enterprise Design:** Complete overhaul of the UI to "Obsidian & Glass" aesthetic.
- **Documentation:** Added `AGENTS.md` and full `/docs` suite.
- **Creative Studio:** Added Alchemy mode for UI auditing and Evolution.
- **Landing Page:** New animated hero section and feature grid.

### Changed
- Refactored `App.tsx` to support the 4-phase generation lifecycle (Plan -> Design -> Build -> Refine).
- Updated `Header.tsx` to a floating island design.
- Improved `ArtifactRenderer` drag-and-drop stability.

### Fixed
- Fixed race conditions in `gemini.ts` stream handling.
- Resolved scroll overflow issues in the properties panel.

## [2.0.0] - 2024-03-15
### Added
- Initial support for Gemini 1.5 Pro.
- Code View with Syntax Highlighting.

## [1.0.0] - 2024-01-01
- Initial Release.
